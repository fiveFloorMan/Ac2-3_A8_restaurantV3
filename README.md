# Restaurant Version three

## 程式的目
* 記錄下所有好吃的餐廳

## 功能 
* 查看所有的餐廳
* 加入新的喜愛餐廳
* 編輯你的餐廳

## 使用方式
* Step1: 下載程式碼
  url: ```https://github.com/fiveFloorMan/Ac2-3_A7_restaurantV2```

* Step2: 使用git bush
  cd 進入資料夾後, 輸入```npm run dev```即可
